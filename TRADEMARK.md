# Trademark policy

This project is licensed under Apache-2.0.

Apache-2.0 does not grant trademark rights. This document clarifies how the project name and branding may be used.

## What is covered

The following are trademarks of the project maintainers (whether registered or unregistered):

- Project names: Cue, CueStack, cue-console, cue-command, cue-mcp, cueme, cuemcp
- Logos, icons, and other brand assets

## Allowed uses (no permission needed)

You may use the names to:

- Refer to the project in documentation, blog posts, reviews, or tutorials.
- Accurately describe that your software is built with or compatible with this project.
- Use unmodified logos in articles that discuss the project (non-commercial or commercial), as long as there is no implication of endorsement.

## Restricted uses (permission required)

You must get permission to:

- Use the names or logos in a way that suggests endorsement, sponsorship, or affiliation.
- Use the names or logos for a product/company name, domain name, or social account that could cause confusion.
- Distribute a modified version of the project and use the original name/logo without clearly distinguishing it.

## Forks

Forks are welcome.

If you distribute a fork, please choose a name and branding that do not imply the fork is the official project.

## Contact

Open an issue in the relevant repository to discuss trademark usage.
